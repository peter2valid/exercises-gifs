PRODUCT DEFINITION
Core Idea

A mobile-first, offline-first workout system that guides users through structured workouts and allows instant set logging in real gym conditions.

Primary Action

Start Workout -> Log Set

Core Flow

Open App -> Start Workout -> Perform Exercise -> Log Set -> Rest -> Repeat -> Finish -> Reward

Target Users
Beginners (primary)
Intermediate gym users
Gym members using shared systems
Core Principles
Action over browsing
Zero friction
Works without internet
One primary action per screen
